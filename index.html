Welcome

Hi — welcome to my POE PARt 3 assignment. It's a simple static project that demonstrates HTML structure, responsive CSS, and client-side JavaScript (forms, a map, and a small gallery). I kept the code readable and commented so it's easy to follow.

What you can try here

Below are the main interactive features I implemented — try them out and see the code in the assets folder.

  • Enquiry form — shows a cost/availability response.
  • Contact form — builds a mailto: so you can open it in your email client.
  • Events list — client-side search/filter with JSON loading.
  • Leaflet map — simple location marker.
  • Gallery with a lightweight lightbox.
